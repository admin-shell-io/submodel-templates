# Handover Documentation (Version 2.0.1)

This is a Submodel template specification for the Asset Administration Shell.

## About this version

This version 2.0.1 is a bug fix release that addresses GitHub issues identified in version 2.0 of the Handover Documentation submodel template.

## Changes in Version 2.0.1

### Bug Fixes

This version addresses the following GitHub issues:

- [#196](https://github.com/admin-shell-io/submodel-templates/issues/196) - Handover Documentation V2: Wrong Language in Display Name for Submodel Element ClassName - The Submodel Element with idShort "ClassName" has set "en" two times in the DisplayName, which makes the Submodel invalid. The language of the LangString with text "Klassenname" should be changed to "de"
- [#147](https://github.com/admin-shell-io/submodel-templates/issues/147) - HandoverDocumentation 2.0: Typos in specification document - In the HandoverDocumentation v2.0 document, there is a number of typos: page 5: "Machinenrichtlinie" instead of "Maschinenrichtlinie", page 19: Missing quotation mark after "use"

## Status

This version is ready for productive use and addresses all known issues from version 2.0.
