# Contact Information (Version 1.0.1)

This is a Submodel template specification for the Asset Administration Shell.

## About this version

This version 1.0.1 is a bug fix release that addresses GitHub issues identified in version 1.0 of the Contact Information submodel template.

## Changes in Version 1.0.1

### Bug Fixes

This version addresses the following GitHub issues:

- [#189](https://github.com/admin-shell-io/submodel-templates/issues/189) - Proposal: Fix non-uniform directory structure for SMT ContactInformation
- [#188](https://github.com/admin-shell-io/submodel-templates/issues/188) - [IDTA-02002-1-0 Contact Information] semanticId in template files for SMC "IPCommunication" does not match specification
- [#98](https://github.com/admin-shell-io/submodel-templates/issues/98) - 02002 Contact Information - Specification does not match the template
- [#97](https://github.com/admin-shell-io/submodel-templates/issues/97) - 02002 Contact Information - Misleadig note for idShort of ContactInformation

## Status

This version is ready for productive use and addresses all known issues from version 1.0.
